# ProtogenOS
C++ and Assembly System (protogen uwu)

```Installation for Windows: 
-------------
git Edouard127/ProtogenOS
cd ./ProtogenOS
curl -O https://qemu.weilnetz.de/w64/2021/qemu-w64-setup-20211215.exe
START /WAIT qemu-w64-setup-20211215.exe
cd ./gnu-efi
make bootloader
cd ../kernel
make kernel
make buildimg
./run2.bat```
